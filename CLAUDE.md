# AmplifyingCareers Website - Project Rules

## Project Overview
This is the website for AmplifyingCareers, a psychology-driven career coaching business run by Jimmy Winfield. The goal is to attract coaching clients, build credibility, and host monthly interview blog posts with industry professionals.

## Brand Guidelines
- Primary green (dark): #2b6e47 — used for nav, section backgrounds, buttons
- Primary green (bright): #3aaa1e — used for logo text and accent highlights
- Background light: #F5F2EE — used for alternating sections and page backgrounds
- Text on dark green sections: white
- Text on light sections: dark green #2b6e47
- Font: Inter (Google Fonts) — bold for headings, regular for body
- Logo file: brand_assets/logo.png

## Design Principles
- Always invoke the frontend-design skill before writing any frontend code, every session, no exceptions
- Modern, clean, and professional — not generic or template-looking
- Psychology-driven and human — warm but credible
- Mobile responsive across all pages
- Use the screenshot loop to self-review and iterate visually after building each section

## Site Structure
- Home: hero, about snapshot, services overview, testimonials, CTA
- Services: Career Change Program (12 sessions) and Career Entry Program (4 sessions)
- About: Jimmy's story, credentials, photo
- Blog: interview posts with industry professionals, added monthly
- Contact: contact form forwarding to jimmy@amplifyingcareers.com, LinkedIn link
